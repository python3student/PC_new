## 动态推送延迟

:::tip
该问题目前仅限动态推送，直播推送无论有多少位延迟都在 10 秒以内。
:::

受限于 B站 对 api 爬取频率限制，目前 HarukaBot 会将需要推送的所有 UP主 排进一个队列，每隔十秒检查一位。因此假设 HarukaBot 订阅了 x 位 UP主，最高延迟就是 10x 秒。计划在 v2 中通过登录账号提高爬取效率。
