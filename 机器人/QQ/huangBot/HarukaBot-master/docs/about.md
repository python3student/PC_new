#  关于

HarukaBot 是我学习 Python 过程中诞生的一个实践项目。初衷只是减轻粉丝群管理的负担，不用再手动一个一个群去通知开播了。

如今随着这个项目不断的完善，我希望 HarukaBot 可以适用于更多的场景（私人订阅、聊天娱乐群），为不同用户提供最合适的推送方式。
